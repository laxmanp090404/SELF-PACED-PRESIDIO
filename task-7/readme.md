# CSS-Only Image Carousel

A simple, responsive image carousel .

## Features

- **Responsive**
- **Smooth Transitions**
- **Navigation Controls**

## How It Works

1. **Radio Buttons**: Hidden `<input type="radio">` elements track the state of the active slide.
2. **Sibling Combinators**: The `~` selector changes the CSS properties of the slides based on which radio button is checked.
