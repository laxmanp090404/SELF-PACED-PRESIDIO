# Complex Responsive Layout with Grid and Flexbox

This project implements a complex, responsive layout using CSS Grid and Flexbox. It involves layout techniques such as overlapping grid items, sticky positioning, and responsive adjustments for different screen sizes.

## Features

- **CSS Grid Layout**: The main content area utilizes a multi-column and multi-row grid structure to organize content boxes.
- **Overlapping Elements**: Demonstrates how to position grid items to overlap each other (e.g., `.overlapbox` spans rows occupied by other elements).
- **Sticky Positioning**: The `.flexedbox` element is set to `position: sticky`, causing it to stick to the top of the viewport while scrolling.
- **Flexbox**: Used within grid items and for the overall page structure (header, footer) to manage alignment and distribution of space.
- **Responsive Design**:
  - **Desktop**: A 3-column layout.
  - **Tablet (max-width: 768px)**: Adapts to a 2-column layout.
  - **Mobile (max-width: 600px)**: Stacks elements in a single column for readability.

## Technologies Used

- **HTML5**: Semantic structure for the page content.
- **CSS3**: Grid Layout, Flexbox, Media Queries, and sticky positioning.
