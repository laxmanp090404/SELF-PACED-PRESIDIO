# HomeSpace Landing Page

This is a Static Webpage Layout with semantic tag usage.

## Features

- **Responsive Design**: Adapts to different screen sizes, including mobile devices.
- **Semantic tags usage**: For better SEO and accessibility.

## Components
- **Header**: Contains the logo and a navigation bar with hover effects.
- **Hero Section**: Features a catchy headline, descriptive text, a "Get Started" call-to-action button, and a hero image.
- **Footer**: Includes a brief description, quick links, contact information, and copyright notice.

## Technologies Used

- **HTML5**: For the structure and semantic elements of the page.
- **CSS3**: For styling, layout (Flexbox), responsive design (Media Queries).

## File Structure

- `index.html`: The main HTML file containing the page structure.
- `style.css`: The CSS file containing all the styles and media queries.
- `assets/`: Directory containing images (logo, icons, hero image).
