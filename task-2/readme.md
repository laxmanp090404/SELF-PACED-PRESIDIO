# CSS Card Component with hover effect

This project displays a responsive anime card component designed to showcase anime details, including a poster image, title, and synopsis.

## Features

- **Responsive Design**: The card layout adapts to different screen sizes. On smaller screens (max-width: 768px), the layout switches from horizontal to vertical.
- **Interactive Hover Effects**:
  - Scales up slightly (`transform: scale(1.05)`).
  - Displays a neon glow shadow (`box-shadow`).
  - Changes border color.
- **Flexbox Layout**: Utilizes CSS Flexbox for centering the component on the page and aligning internal elements.

## Technologies Used

- **HTML5**: Structural markup.
- **CSS3**: Styling, Flexbox, Transitions, and Media Queries.

## File Structure

- `index.html`: The structure of the web page.
- `style.css`: The styling rules.
- `assets/`: Contains images used in the project.
